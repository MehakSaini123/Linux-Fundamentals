# DevOps Principles

## History
Waterfall follows sequential phases. Agile introduced iterative development. DevOps extends Agile by integrating Development and Operations with automation and collaboration.

## Three Ways
1. **Flow** - Improve work from development to production.
2. **Feedback** - Fast feedback to detect and fix issues.
3. **Continual Learning** - Continuous improvement through experimentation.

## CI/CD Lifecycle
Plan → Code → Build → Test → Release → Deploy → Operate → Monitor

## Benefits
- Faster delivery
- Better quality
- Improved collaboration
- Reduced deployment risk
